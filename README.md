
# NodeJS-CronJob

Cron Job Basic, Send Mail by CronJob in NodeJS


## Documentation

[Cron Tab](https://crontab.guru/)

## Installation

```bash
  cd my-project
  npm install
  create .env
  copy env-example
```

## Running the App

```bash
  npm start
```
